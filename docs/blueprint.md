# **App Name**: Family Cookbook

## Core Features:

- Add Recipe: Add a new recipe with details like name, source, time, servings, ingredients, instructions, picture, and freezing suitability.
- Edit/Delete Recipe: Edit or delete existing recipes.
- View Recipe: View recipes with all details, including ingredients, instructions, and pictures.
- Double Recipe: Double the ingredient amounts in a recipe.
- Shopping List: Add ingredients with doubled amounts to a shopping list.
- Print Recipe: Generate a printer-friendly version of the recipe.

## Style Guidelines:

- Primary color: Soft terracotta (#E07A5F) for a warm, inviting feel.
- Background color: Light beige (#F8F5F1), offering a clean, neutral backdrop.
- Accent color: Dusty rose (#BE95C4) for interactive elements, providing a touch of elegance.
- Headline font: 'Belleza', a humanist sans-serif for headlines and short chunks of text. Body font: 'Alegreya', a humanist serif, is used for longer blocks of recipe instructions to ensure readability.
- Use simple, line-based icons related to cooking and food items.
- Clean and organized layout with clear sections for recipe details, ingredients, and instructions.
- Subtle transitions and animations when navigating between recipes.
